Project for CSE115L by Rakin, Omar, Tamim
Project Members [Sk. Rakinul Haque 2532182642, Tawfique Omar Tazwar 2533259642, Tamim Sadiq Sneho 2531929642]

Project Description—This program can only be accessed by students of the class; professors and instructors will have administrative permissions. The program will mainly feature a student resource function that can be used to log books, courses, credits, exams, and dates. Additionally, it will include a calculations section where users can use a multiline scientific calculator and check for different number types. After the user logs out, the resource files are saved under the username, and the information can be retrieved upon login.

Project Parameters—
- Login System
- Logout System
- Administrative Permissions
- Student Resources (logging tools)
- Calculation and number identification tools
- Possible miscellaneous additions
